===
Top
===

Overview
========

Top page of this service.
Center of this page shows an input area for URI.
Users can generate shorten URI by inputing own URI.

URL shoter
==========

Generated URI can be copied to the clip bord by clicking a 'copy' button.
After generated shorten URI, the cursor will automatically select the area
of the URI so that users can copy it quickly.
(Show a message "Press 'Ctrl+C'")

New generated URIs will insert to field under the input area one by one.
These URIs continue to be displayed until the user deletes the Cookie.
