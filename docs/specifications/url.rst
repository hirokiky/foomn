===
URL
===

overview
========

Allowed type of URLs for foomn.


max length: 1000

Allowed schema:

* ftp://
* http://
* https://

Not support:

* localhost
* User and password
* IPAddress v4/v6

Ignoring hosts to prevent infinite redirecting:

* foo.mn
* bit.ly
* tinyurl.com
* goo.gl
* t.co
* ow.ly
* htn.to
* p.tl
